# 🔐 Credential Vault / Password Vault

A secure credential management system built with **ASP.NET Core Minimal API** and **React + TypeScript**.

---

## 📌 Features

* JWT Authentication
* Refresh Token Support
* Password Encryption / Decryption
* CRUD operations for credentials
* Filtering & Search
* Global Exception Handling

---

## 🛠️ Tech Stack

**Backend:** ASP.NET Core Minimal API, EF Core, SQL Server, JWT, Cryptography
**Frontend:** React, TypeScript

---


---

## ⚠️ Highlights

* Encrypted password storage
* Secure token-based authentication
* Clean API structure with global exception handling
